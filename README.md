# CV de Juan Camilo Osorio Oviedo (osvo)

Este repositorio contiene el código fuente del currículum en línea de Juan Camilo Osorio Oviedo, una página web interactiva diseñada para simular un entorno de escritorio con ventanas de terminal.

## ¿Cómo ver el sitio web?

Puedes ver el sitio web en vivo en: [osvo.github.io](https://osvo.github.io)

## Tecnologías utilizadas

Este proyecto fue construido utilizando únicamente tecnologías web estándar:

- HTML
- CSS
- JavaScript
